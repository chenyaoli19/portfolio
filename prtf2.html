Surface Magazine

A mobile publishing Magazine



This is a demo project I worked on using the mobile publishing platform-Pugpig. Pugpig is a platform that gives mobile publishing team the tools they need to create beautiful edition based apps at low cost.


I designed and built this e-magazine using Pugpig's SDK.

Design

HTML5 & CSS3 Supported Mobile Magazine

It's a fun experience building this magazine app from scratch. I designed the layout, chose the content from one of my favorite magazines surface, and utilized new features in HTML5 and CSS3 to make this interactive and sleek demo magazine app.

Features include:

  • Semantic HTML5 Markup
  • CSS3 Animation/Transitions
  • Media Queries
  • HTML5 Vedio Support
  • JQuery lightbox/image slider
  • Responsive Web Design
